# API Documentation

This application uses a GraphQL API provided by Blockware Solutions for accessing marketplace data including mining equipment offers, hosting sites, and deal analytics.

**API Type:** GraphQL  
**Endpoint:** `https://dev.marketplace.api.blockwaresolutions.com/api/opengraphql`  
**Protocol:** HTTPS  
**Format:** JSON
